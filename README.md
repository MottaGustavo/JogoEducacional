# JogoEducacional

>Jogo educacional de perguntas criado para crianças de 4 a 5 anos, o tema: História e curiosidades de Brusque, com linguagem audiovisual adequado para o público que sera infantil, com acessibilidade de reprodução de áudio do texto.
