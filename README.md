# JogoEducacional

> Jogo educacional de perguntas criado para crianças de 4 a 5 anos, o tema: História e curiosidades de Brusque - SC, com linguagem audiovisual adequado para o público infantil, acessibilidade em reprodução de áudio do texto pensado nas crianças que estão aprendendo a ler.


## Como jogar 

1- Abra o arquivo 'html/index.html' no navegador.
2- Leia ou ouça a pergunta (ao apertar o botão laranja com icone de som, o texto da página é reproduzido em voz alta em português)
3- Clique na resposta correta ou não hehe... (Pensado em não causar bloqueio: o usuário sempre passa de "fase" acertando ou não, mas após a opção selecionada é apresentado um feedback gentil )
4- Acompanhe sua evolução pela barra de progresso (1 de 15)
5- Responda todas as 15 perguntas e veja seu resultado final!

## Como Rodar o Projeto
 
```bash
# 1. Clone o repositório
git clone https://github.com/MottaGustavo/JogoEducacional.git
 
# 2. Acesse a pasta
cd JogoEducacional
 
# 3. Abra no navegador
# Dê dois cliques em html/index.html
# ou use a extensão Live Server no VS Code
```
